# Assignment 12 - Function Bindings

## Goal

Use Azure Function output binding to save IoT location data into Azure Blob Storage.

## Files

- `code/store_location/__init__.py`
- `code/store_location/function.json`
- `report.md`
- `screenshots/`

## Note

The binding uses `AzureWebJobsStorage`. Set this value in `local.settings.json` or Azure App Settings. Do not commit the real storage connection string.
